# Santos-Modelados-e-Impresos-en-Recina-3D
Diseño y fabricación de figuras religiosas representativas mediante impresión 3D en resina. Las piezas destacan por su alta precisión, nivel de detalle y acabado, siendo ideales para uso conmemorativo y decorativo.
